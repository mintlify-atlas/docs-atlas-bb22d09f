AI Engineering from Scratch is an open-source course that takes you from first principles all the way to production-ready AI systems. Every lesson produces something you can actually use: a prompt, a Claude Code skill, a deployable agent, or an MCP server. By the end you have a toolkit, not just a certificate.

## What the course covers

The course is organized into 20 phases spanning roughly 290 hours of hands-on work. It covers the complete AI engineering stack in four broad arcs.

## Who this is for

The course is designed for anyone who writes code and wants to understand how AI actually works — not just call APIs. There is no single entry point; you start where your existing knowledge ends.

| You are... | Start at... | Estimated time |
|---|---|---|
| New to programming and AI | Phase 0: Setup & Tooling | ~290 hours |
| Know Python, new to ML | Phase 1: Math Foundations | ~270 hours |
| Know ML, new to deep learning | Phase 3: Deep Learning Core | ~200 hours |
| Know deep learning, want LLMs and agents | Phase 10: LLMs from Scratch | ~100 hours |
| Senior engineer who wants to build agents | Phase 14: Agent Engineering | ~60 hours |

## What makes this course different

Most AI courses give you one slice — NLP, or vision, or agents — in Python only, at a surface level, with passive video content. This course takes a different approach on every one of those axes.

**Full-stack scope.** The curriculum covers everything from linear algebra through autonomous agent swarms in a single continuous sequence. You don't have to stitch together five separate courses.

**Build it first, then use the framework.** Every major concept is implemented from scratch before a framework is introduced. You build backpropagation from scratch in Phase 3, then see how PyTorch does it in the same phase. You build the agent loop from scratch in Phase 14, then compare it to production agent frameworks. You understand what the abstraction does because you built the thing it abstracts.

**Four languages.** Python for modeling and ML, TypeScript for agent tooling and MCP servers, Rust for performance-critical components, and Julia for numerical computing. Lessons use the language that fits the domain.

**Reusable output at every lesson.** Other courses end with "congratulations, you learned X." Every lesson here ends with a reusable artifact saved to `outputs/`: a prompt you can paste into any AI assistant, a Claude Code skill you can install, an agent definition, or an MCP server. By Phase 19 you have a complete toolkit.

**AI-native learning.** The course includes two built-in Claude Code skills — `/find-your-level` for placement and `/check-understanding <N>` for per-phase quizzes — that test you as you go. You learn AI _with_ an AI agent, not just about AI.
