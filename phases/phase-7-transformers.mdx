## Core concept: self-attention from scratch

Self-attention lets every position in a sequence attend to every other position in a single parallel step. This is the lesson that is already complete in the course — the code below is taken directly from the source.

The key formula: `Attention(Q, K, V) = softmax(Q @ K^T / sqrt(dk)) @ V`

```python
import numpy as np

def softmax(x):
    shifted = x - np.max(x, axis=-1, keepdims=True)
    exp_x = np.exp(shifted)
    return exp_x / np.sum(exp_x, axis=-1, keepdims=True)

def scaled_dot_product_attention(Q, K, V):
    dk = Q.shape[-1]
    scores = Q @ K.T / np.sqrt(dk)   # scale prevents softmax saturation
    weights = softmax(scores)         # (n, n) probability distribution
    output = weights @ V              # weighted sum of value vectors
    return output, weights

class SelfAttention:
    def __init__(self, d_model, dk, dv, seed=42):
        rng = np.random.default_rng(seed)
        scale_qk = np.sqrt(2.0 / (d_model + dk))
        self.Wq = rng.normal(0, scale_qk, (d_model, dk))
        self.Wk = rng.normal(0, scale_qk, (d_model, dk))
        scale_v = np.sqrt(2.0 / (d_model + dv))
        self.Wv = rng.normal(0, scale_v, (d_model, dv))

    def forward(self, X):
        Q = X @ self.Wq   # (n, dk) — "what am I looking for?"
        K = X @ self.Wk   # (n, dk) — "what do I contain?"
        V = X @ self.Wv   # (n, dv) — "what do I offer?"
        return scaled_dot_product_attention(Q, K, V)
```

### How the attention matrix reads

```
              k1    k2    k3    k4    k5
        +-----+-----+-----+-----+-----+
   q1   | 2.1 | 0.3 | 0.1 | 0.8 | 0.2 |   ← how much token 1 attends to each key
        +-----+-----+-----+-----+-----+
   q2   | 0.4 | 1.9 | 0.7 | 0.1 | 0.3 |
        +-----+-----+-----+-----+-----+
   q3   | 0.2 | 0.6 | 2.3 | 0.5 | 0.1 |
        +-----+-----+-----+-----+-----+

Each row sums to 1.0 after softmax.
High diagonal = each token attends mostly to itself.
Off-diagonal weights capture cross-token relationships.
```

### Why divide by sqrt(dk)?

Dot products grow with dimension. At `dk = 64`, raw scores reach values that push softmax into saturation — a region where all gradients vanish. Dividing by `sqrt(dk)` normalizes the variance and keeps softmax in a useful gradient range.

### Multi-head attention

Multi-head attention runs `n_heads` parallel self-attention functions, each with its own learned projections of size `dk = d_model / n_heads`, then concatenates and projects the results. This lets the model simultaneously attend to different kinds of relationships — syntactic structure, coreference, semantic similarity — in the same forward pass.

```python
class MultiHeadSelfAttention:
    def __init__(self, d_model, n_heads, seed=42):
        assert d_model % n_heads == 0
        self.dk = d_model // n_heads
        self.heads = [
            SelfAttention(d_model, self.dk, self.dk, seed=seed + i)
            for i in range(n_heads)
        ]
        rng = np.random.default_rng(seed + n_heads)
        scale = np.sqrt(2.0 / (d_model * 2))
        self.Wo = rng.normal(0, scale, (n_heads * self.dk, d_model))

    def forward(self, X):
        outputs, weights = zip(*[h.forward(X) for h in self.heads])
        concatenated = np.concatenate(outputs, axis=-1)
        return concatenated @ self.Wo, list(weights)
```
