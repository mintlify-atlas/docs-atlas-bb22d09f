Reinforcement learning is the framework in which an agent learns by interacting with an environment — taking actions, receiving rewards, and updating its policy to maximize long-term return. RL is the foundation of RLHF (Reinforcement Learning from Human Feedback), the technique used to align language models like ChatGPT and Claude. This phase builds every algorithm from the Bellman equation up through PPO and reward modeling, all in Python, against OpenAI Gym and custom environments.

## Core concept: Q-learning and the Bellman equation

Q-learning is the simplest and most instructive RL algorithm. It maintains a table of Q-values `Q(s, a)` — the expected cumulative reward for taking action `a` in state `s` and then following the optimal policy. The Bellman equation provides the update rule.

```python
import numpy as np

class QLearningAgent:
    """Tabular Q-learning with epsilon-greedy exploration."""

    def __init__(self, n_states, n_actions, lr=0.1, gamma=0.99,
                 epsilon_start=1.0, epsilon_end=0.01, epsilon_decay=0.995):
        self.Q = np.zeros((n_states, n_actions))
        self.lr = lr
        self.gamma = gamma            # discount factor
        self.epsilon = epsilon_start
        self.epsilon_end = epsilon_end
        self.epsilon_decay = epsilon_decay
        self.n_actions = n_actions

    def select_action(self, state):
        """Epsilon-greedy: explore randomly or exploit best known action."""
        if np.random.rand() < self.epsilon:
            return np.random.randint(self.n_actions)
        return np.argmax(self.Q[state])

    def update(self, state, action, reward, next_state, done):
        """Bellman update: Q(s,a) ← Q(s,a) + α [r + γ max_a' Q(s',a') - Q(s,a)]"""
        td_target = reward + (0 if done else self.gamma * np.max(self.Q[next_state]))
        td_error  = td_target - self.Q[state, action]
        self.Q[state, action] += self.lr * td_error
        self.epsilon = max(self.epsilon_end, self.epsilon * self.epsilon_decay)
        return abs(td_error)
```

### From Q-learning to RLHF

The RLHF pipeline used to align language models follows the same RL loop, with three key changes:

| RL concept | RLHF equivalent |
|------------|----------------|
| Environment state | Current conversation context + partial response |
| Action | Next token sampled from the language model |
| Reward signal | Score from a reward model trained on human preference pairs |
| Policy update | PPO with KL penalty to prevent the model drifting too far from the supervised baseline |
