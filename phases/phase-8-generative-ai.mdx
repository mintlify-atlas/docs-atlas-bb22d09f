Generative AI is the field of building models that produce new content — images, video, audio, and 3D geometry — rather than merely classifying existing data. This phase traces the full genealogy of generative modeling: from the latent-variable perspective of autoencoders, through the adversarial training of GANs, to the denoising process of diffusion models that now power most state-of-the-art image synthesis. Every architecture is built from scratch before being used through existing libraries.

## Core concept: DDPM forward and reverse process

Diffusion models learn to reverse a gradual noising process. The forward process adds Gaussian noise over `T` timesteps until the image is pure noise. A neural network (typically a U-Net) learns to predict and remove that noise one step at a time.

```python
import numpy as np

class DDPMScheduler:
    """Linear noise schedule for DDPM (Ho et al., 2020)."""

    def __init__(self, T=1000, beta_start=1e-4, beta_end=0.02):
        self.T = T
        # Linear schedule: beta_t increases from beta_start to beta_end
        self.betas = np.linspace(beta_start, beta_end, T)
        self.alphas = 1 - self.betas
        self.alpha_bars = np.cumprod(self.alphas)   # ᾱ_t = ∏ α_s for s≤t

    def q_sample(self, x0, t, noise=None):
        """Forward process: add noise to x0 at timestep t.

        Closed form: x_t = sqrt(ᾱ_t) * x0 + sqrt(1 - ᾱ_t) * ε
        """
        if noise is None:
            noise = np.random.randn(*x0.shape)
        alpha_bar_t = self.alpha_bars[t]
        return (np.sqrt(alpha_bar_t) * x0 +
                np.sqrt(1 - alpha_bar_t) * noise), noise

    def p_sample(self, x_t, t, predicted_noise):
        """Reverse process: one denoising step given predicted noise.

        x_{t-1} = (1/sqrt(α_t)) * (x_t - β_t/sqrt(1-ᾱ_t) * ε_θ) + σ_t * z
        """
        beta_t = self.betas[t]
        alpha_t = self.alphas[t]
        alpha_bar_t = self.alpha_bars[t]

        coeff = beta_t / np.sqrt(1 - alpha_bar_t)
        mean = (x_t - coeff * predicted_noise) / np.sqrt(alpha_t)

        if t > 0:
            noise = np.random.randn(*x_t.shape)
            variance = np.sqrt(beta_t)
        else:
            noise = 0
            variance = 0

        return mean + variance * noise
```

The elegant insight: instead of learning to generate directly, the model learns to denoise. At inference time it starts from pure Gaussian noise and takes `T` denoising steps, each guided by the learned U-Net.
