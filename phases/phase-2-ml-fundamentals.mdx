Classical machine learning is not obsolete — it powers the majority of production AI systems in use today. Phase 2 covers the full spectrum of supervised and unsupervised algorithms: from the geometry of linear regression to the ensemble dynamics of gradient boosting, from K-Means clustering to anomaly detection. Every algorithm is built from scratch in pure Python before being re-implemented with scikit-learn, so you understand what the library is doing — and when to reach for something else. By the end you'll have a complete mental model of the ML workflow: from raw data through feature engineering, model selection, hyperparameter tuning, and evaluation pipelines.

## What you'll ship from this phase

- A **linear regression implementation** in ~50 lines of NumPy — gradient descent and normal equation both, with a comparison
- A **Random Forest from scratch** that shows how bootstrapped trees combine for lower-variance predictions
- A **model evaluation toolkit** (confusion matrix, ROC-AUC, k-fold CV) reusable across any binary classification problem
- An **MLflow experiment tracking setup** that logs hyperparameter sweeps and stores the best model artifact

```python
# Lesson 02: Linear Regression from Scratch
# Gradient descent implementation — no scikit-learn

import numpy as np

class LinearRegression:
    def __init__(self, lr=0.01, n_iters=1000):
        self.lr = lr
        self.n_iters = n_iters
        self.weights = None
        self.bias = None

    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0.0

        for _ in range(self.n_iters):
            y_pred = X @ self.weights + self.bias
            dw = (2 / n_samples) * X.T @ (y_pred - y)
            db = (2 / n_samples) * np.sum(y_pred - y)
            self.weights -= self.lr * dw
            self.bias    -= self.lr * db

    def predict(self, X):
        return X @ self.weights + self.bias

    def mse(self, X, y):
        return np.mean((self.predict(X) - y) ** 2)

# Usage
X = np.random.randn(100, 3)
y = X @ np.array([2.0, -1.0, 0.5]) + 0.1 * np.random.randn(100)

model = LinearRegression(lr=0.05, n_iters=500)
model.fit(X, y)
print(f"MSE: {model.mse(X, y):.4f}")
print(f"Weights: {model.weights}")
```
