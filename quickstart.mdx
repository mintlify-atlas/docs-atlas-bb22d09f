You can start this course in three different ways depending on how you learn. Just reading works. Cloning and running the code works better. Using the AI placement quiz to find your exact starting point is the recommended path for anyone who already knows some ML or deep learning.

<Tabs>
  <Tab title="Just start reading">
    Browse to any completed phase in the sidebar and start reading. Every lesson has a `docs/en.md` file with the full explanation, diagrams, and code walkthrough.

    Good starting points:

    - **New to AI:** [Phase 1: Math Foundations](/phases/phase-1-math) — linear algebra through optimization, all with Python code
    - **Know ML basics:** [Phase 3: Deep Learning Core](/phases/phase-3-deep-learning) — backpropagation from scratch, then PyTorch
    - **Want to build agents now:** [Phase 14: Agent Engineering](/phases/phase-14-agent-engineering) — the agent loop, tool dispatch, memory, and subagents

    <Note>
    The reading path gives you the concepts but not the practice. The lessons are structured for active work — there are exercises and a build-from-scratch section in every lesson.
    </Note>
  </Tab>
  <Tab title="Clone and run">
    Clone the repository and run code from the first lesson to verify your environment is set up correctly.

    <Steps>
      <Step title="Clone the repository">
        ```bash
        git clone https://github.com/rohitg00/ai-engineering-from-scratch.git
        cd ai-engineering-from-scratch
        ```
      </Step>
      <Step title="Set up a Python environment">
        The course uses Python 3.10+. Create a virtual environment to keep dependencies isolated.

        ```bash
        python -m venv .venv
        source .venv/bin/activate   # Windows: .venv\Scripts\activate
        pip install numpy matplotlib
        ```
      </Step>
      <Step title="Run your first lesson">
        Start with Phase 1, Lesson 1 — linear algebra intuition. This file has no external dependencies beyond NumPy.

        ```bash
        python phases/01-math-foundations/01-linear-algebra-intuition/code/vectors.py
        ```

        If this runs without errors, your environment is ready for Phases 0 through 3. Later phases add dependencies (PyTorch, Anthropic SDK, etc.) per lesson.
      </Step>
      <Step title="Explore the outputs">
        Each lesson saves reusable artifacts to its `outputs/` directory. For example, after working through the loss functions lesson:

        ```bash
        ls phases/03-deep-learning-core/05-loss-functions/outputs/
        # prompt-loss-function-selector.md
        # prompt-loss-debugger.md
        ```

        These prompts work immediately — paste them into Claude, ChatGPT, or any AI assistant to get expert-level help on loss function selection and debugging.
      </Step>
      <Step title="Check understanding after each phase">
        After completing a phase, quiz yourself using the built-in Claude Code skill:

        ```bash
        # In Claude Code, after finishing Phase 3:
        /check-understanding 3
        ```

        This generates 8 questions from the actual lesson content — 4 conceptual and 4 practical — and tells you which lessons to revisit based on your answers.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Find your level (recommended)">
    If you already know some ML, deep learning, or LLM engineering, don't start from Phase 1. Use the placement quiz to skip phases you've already mastered.

    <Steps>
      <Step title="Open Claude Code in the repo directory">
        ```bash
        git clone https://github.com/rohitg00/ai-engineering-from-scratch.git
        cd ai-engineering-from-scratch
        # Open Claude Code here
        ```
      </Step>
      <Step title="Run the placement skill">
        ```bash
        /find-your-level
        ```

        Claude Code will ask you 10 multiple-choice questions across 5 knowledge areas: math and statistics, classical ML, deep learning, NLP and transformers, and applied AI (RAG, agents).
      </Step>
      <Step title="Review your personalized path">
        After the quiz, you receive a score breakdown and a full 20-phase table showing which phases to skip, which to review, and which to complete — with hour estimates for your actual path.

        Example output for a score of 6/10:

        ```
        Math & Statistics:    2/2
        Classical ML:         2/2
        Deep Learning:        1/2
        NLP & Transformers:   1/2
        Applied AI:           0/2

        Entry point: Phase 7 — Transformers Deep Dive
        Your personalized path: ~130 hours across 12 phases.
        ```
      </Step>
      <Step title="Start at your entry phase">
        Jump directly to the recommended entry phase. The quiz identifies specific lessons to review in areas where you scored 1/2 rather than 2/2.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Prerequisites

The course has minimal hard requirements. What matters is your intent.

**You need:**
- The ability to write code in any language (Python strongly preferred for early phases)
- A working Python 3.10+ environment
- The willingness to build things from scratch before reaching for frameworks

**You do not need:**
- A math or computer science degree
- Prior ML experience (Phase 0 and Phase 1 cover what you need)
- A GPU (most lessons through Phase 9 run on CPU; later phases note when GPU helps)

**For Phase 14+ (agents and MCP):**
- An Anthropic API key for lessons that call Claude
- Node.js 18+ for TypeScript lessons

<Info>
TypeScript is used in Phases 13–16 for agent tooling and MCP server development. If you only know Python, these lessons include enough TypeScript context to follow along — but installing Node.js beforehand helps.
</Info>
